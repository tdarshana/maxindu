A Template for develop ES6 web application

In this article we will get to know about how to configure newest ES6 script on your local machine along with transpilers for HTML and CSS.


For development of web applications using ECMA Script 6 we can use transpilers that transpile your Javascript code into ECMA Script 5. For this purpose we use Babel tranpiler. And also we use SCSS transpiler to convert SCSS code into browser understandable CSS code. Also we use Jade templating machine for easy HTML coding.


Here we use Grunt as our major task runner. By creating Grunt watch on change task we reduced manual integration of compile the code again and again. If you need more details on grunt task please refer getting-started guide on gruntjs.com.


To configure this code & start your application develop from this code skeliton first clone this github repo to your local machine.

git clone "https://github.com/MaxIndu/dev_jade_es6_scss_watch_grunt.git"
Then navigate into that directory using terminal (command prompt on windows). Then enter the following to install all the npm dependancies that is required to run the code correctly.

npm install
Then to watch changes run grunt task as follows.

grunt watch
Now you can edit codes in src directory. When you save them grunt will automatically compile them into relevent formats. You can edit files in the directories "src/jade", "src/js", "src/scss".
