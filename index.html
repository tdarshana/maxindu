A Template for develop ES6 web applications

In this article we will get to know about how to configure newest ES6 script development environment on your local machine along with transpilers for HTML and CSS.


For development of web applications using ECMA Script 6 we can use transpilers that transpiles your ES6 Javascript code into ECMA Script 5. For this purpose we use Babel transpiler. And also we use SCSS transpiler to convert SCSS code into browser understandable CSS code. Also we use Jade templating machine for easy HTML coding.
